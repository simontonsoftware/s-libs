    Class InjectableSuperclassAbstract

    Use as the superclass for anything managed by Angular's dependency injection for care-free use of subscribeTo(). It simply calls unsubscribe() when the object is destroyed. Note that the object must be created in an injection context.

    @Injectable() // or @Component(), @Directive() or @Pipe(), but consider DirectiveSuperclass
    class MyThing extends InjectableSuperclass {
      constructor(somethingObservable: Observable) {
        super();
        this.subscribeTo(somethingObservable);
      }
    }
